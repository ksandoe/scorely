# Scorely

Social music rating app scaffold.

## Repo layout

- `client/` React PWA (Vite)
- `api/` Express API (Node)
- `supabase/` SQL migrations + seeds
- `docs/` PRD, site map, OpenAPI, deployment notes

## Getting started

### Client

1. `npm install`
2. `npm run dev`

### API

1. `npm install`
2. `npm run dev`

### Supabase

SQL placeholders live under `supabase/migrations` and `supabase/seeds`.
